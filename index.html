HTML & CSS

HTML (Hyper Text Markup Language) არ არის დაპროგრამების ენა, მისი მეშვეობითვებგვერდის სტრუქტურის აწყობა და შემუშავება ხდება. HTML5 არის სტანდარტის უახლესი ევოლუცია, რომელიც განსაზღვრავს HTML- ს. ტერმინი წარმოადგენს ორ განსხვავებულ ცნებას. ეს არის HTML– ის ახალი ვერსია, ახალი ელემენტებით, ატრიბუტებითა და ქცევებით და ტექნოლოგიების უფრო დიდი წყობით, რომელიც საშუალებას გაძლევთ შექმნათ უფრო მრავალფეროვანი და ძლიერი ვებ – გვერდები და პროგრამები. ამ ნაკრს ზოგჯერ HTML5 და მეგობრები ჰქვია და ხშირად შემოკლებით მხოლოდ HTML5 CSS (Cascading Style Sheets) - სტილების კასკადური ცხრილები HTML-ის დამატებაა და მის შესაძლებლობებს მნიშვნელოვნად აფართოებს. CSS ვებდიზაინერს საშუალებას აძლევს, ვებგვერდი უფრო ეფექტური და, დიზაინის თვალსაზრისით, ადვილად მართვადი გახადოს.
